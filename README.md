# Judul Project

**SehatYuk - Aplikasi Perhitungan BMI dan BMR**

---

# Deskripsi Aplikasi

SehatYuk merupakan aplikasi Android sederhana yang dibuat menggunakan bahasa pemrograman Kotlin di Android Studio. Aplikasi ini berfungsi untuk membantu pengguna menghitung:

- Body Mass Index (BMI)
- Basal Metabolic Rate (BMR)

Aplikasi dikembangkan menggunakan konsep **Function** sehingga setiap proses memiliki fungsi masing-masing, membuat kode lebih terstruktur, mudah dipahami, dan mudah dipelihara.

---

# Fitur Aplikasi

- Input Nama
- Input Berat Badan (kg)
- Input Tinggi Badan (cm)
- Input Umur (tahun)
- Validasi Input
- Perhitungan BMI beserta kategori
- Perhitungan BMR menggunakan rumus Mifflin-St Jeor
- Menampilkan hasil perhitungan secara otomatis

---

# Function yang Digunakan

Aplikasi ini menggunakan beberapa function utama, yaitu:

1. **validateInput()**
   - Memvalidasi seluruh input agar tidak kosong.

2. **calculateBMI(weight, height)**
   - Menghitung nilai BMI.
   - Menentukan kategori BMI.

3. **calculateBMR(weight, height, age)**
   - Menghitung kebutuhan kalori basal (BMR).

4. **tampilkanHasil()**
   - Menampilkan hasil perhitungan ke layar aplikasi.
